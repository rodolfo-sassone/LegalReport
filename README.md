# Legal Report
Repository for the Digital Forensics and Biometric course of the University of Padova.

### ndb.bash (no directory blocks)
  usage: ndb.bash input_image.dd \
  It takes in input a disk image and output all the inodes without direct blocks
### checksum.bash
  usage: checksum.bash path/of/the/directory [output.file] \
  It takes in input the directory containing the file u want to compute md5 and sha256 cheksum and optionally the name of the output file (default checksum.txt). Gives in output a file with name, md5 and checksum structured as a latex table.
